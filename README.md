# Suryadev Seeds Admin Application

This is a React Native application for managing e-shopping for a seeds company, built using Expo SDK 54 and TypeScript. The application integrates Firebase for backend services and provides a user-friendly interface for admin management.

## Features

- **Custom Fonts**: The application supports custom fonts located in the `src/assets/fonts` directory.
- **Reusable Components**: Common components such as buttons, headers, and tab bars are located in the `src/components/common` directory.
- **Screens**: The application includes multiple screens:
  - Home Screen
  - Category Screen
  - Order Screen
  - Login Screen
- **Navigation**: The app uses a tab navigator for easy navigation between Home, Category, and Order screens.
- **Firebase Integration**: The application is configured to interact with Firebase services for authentication and data management.
- **Responsive Design**: The app supports both light and dark themes, with deep green buttons for a consistent look.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd suryadev-seeds-admin
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

## Usage

- Open the application on your device or emulator.
- Use the login screen to authenticate.
- Navigate through the app using the bottom tab menu.
